Tryckys RTP (NeoForge 1.21.1)
============================

Simple server-side random teleport mod.

Features (WIP)
--------------
- `/rtp` command
- Per-player cooldown (default: 1 hour)
- Safe-spot search (avoid bad blocks/liquids)
- Configurable radius and attempts

Development
-----------
- Java 21
- NeoForge 1.21.1

Notes
-----
This repository started from the NeoForge MDK template and has been cleaned to remove all example blocks/items/tabs.
