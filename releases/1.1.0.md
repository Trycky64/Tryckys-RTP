## W03–W13 — QOL / UX / CONFIG bundle

### Added
- Arrival title/subtitle (configurable)
- Cooldown actionbar (throttled, no spam) + optional progress bar
- `/rtp status` (cooldown + dimension allowed)
- `/rtp reload` (best-effort config reload + cache clear)
- Detailed `/rtp help`
- Advanced permissions:
  - `tryckysrtp.bypass.cooldown`
  - `tryckysrtp.bypass.unsafe`
  - `tryckysrtp.command.reload`
  - OP-level fallback supported
- Silent mode (`messages.silent`)
- Configurable logging level (OFF / INFO / DEBUG)
- i18n-ready structure (`assets/tryckysrtp/lang/...`)

### Safety / Robustness
- Hard rule: never teleport into liquids (even with bypassUnsafe)
- Config validation warnings (no crashes)
- Cleaner internal caching + safer reload behavior

### Configuration
All options are server-side in `tryckysrtp-common.toml`.

No breaking changes.
