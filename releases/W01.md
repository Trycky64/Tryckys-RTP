## W01 — Messages configurables

### Ajout
- Messages configurables via `tryckysrtp-common.toml` :
  - `messages.success`
  - `messages.fail`
- Placeholders supportés :
  - `%x%`, `%y%`, `%z%`, `%dimension%`, `%player%`

### Compatibilité
- Aucun breaking change.
- Le comportement par défaut reste identique si tu ne modifies pas le fichier de configuration.

### Configuration
Exemple :

```toml
[messages]
success = "RTP: %x% %y% %z% (%dimension%)"
fail = "RTP impossible."
```
